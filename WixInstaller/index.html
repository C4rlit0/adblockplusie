Installation for Adblock Plus for IE

Existing Installation

Directories and Files

Either LOCALLOWFOLDER or AppDataFolder is the location for the HTML tree, depending on OS type.
  • LOCALLOWFOLDER
    • Initialized from the registry to AppDataLow, the low-integrity application data directory.
    • AVASTADBLOCKERFOLDERVISTA
      • Used only for Windows Vista and above.
  • AppDataFolder
    • A Windows Installer property: "The full path to the Roaming folder for the current user."
    • AVASTADBLOCKERFOLDER
      • Used only for Windows XP and below.
This folder contains per-user application data.
  • settings.ini
  • settings_page_w.ini
  • dictionary_w.ini
This folder also contains the HTML tree, which is not user-specific and doesn't really belong in this directory.
  • html
    • static
      • js
      • img
    • templates
